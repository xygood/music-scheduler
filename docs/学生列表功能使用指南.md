## 📊 列排序功能

### 基本排序操作

1. **升序排序**: 点击列头进行升序排序
2. **降序排序**: 再次点击同一列头切换到降序
3. **重置排序**: 第三次点击列头重置为默认排序

### 支持排序的列

- ✅ **学号**: 数字/字符串混合排序
- ✅ **姓名**: 按字母顺序排序
- ✅ **班级**: 按班级编号排序
- ✅ **类型**: 普通班/专升本排序
- ✅ **主项**: 按乐器类型排序
- ✅ **备注**: 按备注内容排序

### 排序状态指示

- 📈 **升序**: 显示向上箭头 ↑ 
- 📉 **降序**: 显示向下箭头 ↓
- 🔄 **重置**: 箭头消失，恢复默认排序

### 使用示例

```
场景: 按学生姓名排序
1. 点击"姓名"列头
2. 姓名按A→Z排序
3. 再次点击"姓名"列头
4. 姓名按Z→A排序
5. 第三次点击"姓名"列头
6. 恢复按学号排序
```

## ⚙️ 列设置功能

### 访问列设置

1. 在表格右上角找到"**列设置**"按钮
2. 点击按钮打开设置面板
3. 在面板中勾选/取消勾选要显示的列

### 可配置列

| 列名 | 默认状态 | 说明 |
|------|---------|------|
| ☐ 选择框 | ✅ 显示 | 批量选择功能 |
| 📚 学号 | ✅ 显示 | 学生学号 |
| 👤 姓名 | ✅ 显示 | 学生姓名 |
| 🏫 班级 | ✅ 显示 | 所属班级 |
| 🏷️ 类型 | ✅ 显示 | 普通班/专升本 |
| 🎵 主项 | ✅ 显示 | 主要专业 |
| 🎶 副项 | ✅ 显示 | 副项专业 |
| 📝 备注 | ✅ 显示 | 备注信息 |
| ⚡ 操作 | ✅ 显示 | 编辑/删除按钮 |

### 列设置面板

```
┌─────────────────────────┐

├─────────────────────────┤
│ ☑️ 学号                  │
```

## ✅ 批量操作功能

### 选择记录

#### 单个选择
1. 点击行首的复选框 ✅
2. 记录被选中，背景变为蓝色

#### 批量选择
1. **全选**: 点击表头的复选框
2. **取消全选**: 再次点击表头复选框
3. **反选**: 取消部分选择后再次全选

### 可用批量操作

#### 1. 批量编辑 📝
- **功能**: 批量修改学生信息
- **状态**: 功能开发中
- **点击提示**: "选中 X 名学生，将在下一个版本中实现此功能"

#### 2. 批量删除 🗑️
- **功能**: 删除选中的学生记录
- **安全**: 操作前显示确认对话框
- **不可撤销**: 删除后无法恢复

**删除确认对话框**:
```
┌─────────────────────────────────┐
│ ❌ 确认删除                      │
├─────────────────────────────────┤
│ 确定要删除选中的 3 名学生吗？      │
│ 此操作不可撤销。                  │
├─────────────────────────────────┤
│  [取消]     [确认删除]           │
└─────────────────────────────────┘
```

#### 3. 批量导出 📤
- **功能**: 将选中的学生信息导出为Excel文件
- **格式**: 保持原有导出格式
- **状态**: 立即可用

### 批量操作栏

选择记录后，表格上方会出现操作栏：

```
┌─────────────────────────────────────┐
│ ✅ 已选择 3 项                      │
│ [批量编辑] [批量删除] [导出选中]      │
└─────────────────────────────────────┘
```

## 🎯 表格交互优化

### 行状态指示

#### 1. 普通状态
- 白色背景
- 悬停时变为浅灰色

#### 2. 选中状态
- 蓝色背景 (`bg-blue-50`)
- 选中复选框高亮显示

#### 3. 悬停效果
- 行背景变为浅灰色
- 操作按钮淡入显示

### 操作按钮

#### 单行操作按钮
- **编辑按钮** ✏️: 蓝色，编辑学生信息
- **删除按钮** 🗑️: 红色，删除学生记录

#### 显示时机
- **默认**: 操作按钮隐藏
- **悬停**: 操作按钮淡入显示
- **选中**: 操作按钮始终可见

### 表格工具栏

#### 记录计数
- **显示格式**: "显示 X / Y 条记录"
- **X**: 当前过滤结果数量
- **Y**: 总学生数量

#### 选择状态
- **无选择**: 不显示
- **有选择**: 显示"已选择 X 项"

#### 2. 错误通知 ❌
- **颜色**: 红色  
- **图标**: ❌

**示例**:

- **颜色**: 蓝色

#### 4. 警告通知 ⚠️
- **颜色**: 黄色
- **图标**: ⚠️

### 通知管理

#### 自动消失
- **默认时间**: 5秒

#### 键盘操作
- **ESC键**: 关闭当前通知

## 🔍 搜索和过滤

### 搜索功能

#### 搜索范围
- **学生姓名**: 支持模糊匹配
- **学号**: 支持精确/模糊匹配
- **主项专业**: 支持乐器名称匹配
- **班级**: 支持班级名称匹配

#### 搜索提示
```
🔍 搜索学生...
├── 输入"张三" → 找到张三同学
├── 输入"2024" → 找到2024级学生
├── 输入"钢琴" → 找到钢琴专业学生
└── 输入"音乐学2401" → 找到特定班级
```

### 过滤条件

#### 1. 专业过滤
- **全部专业**: 显示所有学生
- **钢琴**: 仅显示钢琴专业学生
- **声乐**: 仅显示声乐专业学生
- **其他器乐**: 古筝、小提琴等具体专业

#### 2. 班级过滤
- **全部班级**: 显示所有班级
- **具体班级**: 选择特定班级

#### 3. 类型过滤
- **全部类型**: 显示所有类型
- **普通班**: 仅显示普通班学生
- **专升本**: 仅显示专升本学生

### 过滤状态指示

当应用过滤条件时，界面会显示：
```
📊 已过滤 | 🔍 搜索: "张三" | 🎵 专业: 钢琴 | 🏫 班级: 全部
```

## 🛠️ 故障排除

### 常见问题

#### 1. 排序不生效
**原因**: 数据类型不匹配
**解决**: 确保排序列包含有效数据

#### 2. 批量操作按钮不可用
**原因**: 没有选中任何记录
**解决**: 先选择要操作的学生记录

#### 3. 列设置不保存
**原因**: 浏览器隐私模式
**解决**: 使用正常模式访问

#### 4. 导出文件无法打开
**原因**: 文件格式不兼容
**解决**: 确保使用Excel 2007或更新版本
