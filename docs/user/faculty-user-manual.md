## 什么是教研室？

教研室是按乐器专业划分的教学单位，用于：
- 按专业组织教师和学生
- 确保教师只教授其专长领域的课程
- 统计各专业的工作量和使用率
- 优化教学资源配置

---

## 教研室类型

系统包含三个教研室：

### 钢琴专业 (PIANO)
- **代码**: PIANO
- **包含乐器**: 钢琴、钢琴合奏、钢琴伴奏
- **班级规模**: 每班最多5人
- **适用课程**: 钢琴基础、钢琴进阶、钢琴表演等

### 声乐专业 (VOCAL)
- **代码**: VOCAL
- **包含乐器**: 声乐
- **班级规模**: 每班最多5人
- **适用课程**: 声乐基础、声乐技巧、合唱等

### 器乐专业 (INSTRUMENT)
- **代码**: INSTRUMENT
- **包含乐器**: 古筝、笛子、小提琴、萨克斯、大提琴、古琴、双排键等
- **班级规模**:
  - 古筝、笛子、竹笛、葫芦丝: 每班最多8人
  - 其他乐器: 每班最多5人
- **适用课程**: 各乐器的基础、进阶、合奏等

## 教师资质管理

### 查看我的资质

1. 登录系统
2. 左侧导航点击「教学资质」
3. 查看「我的教学资质」区域

资质状态说明：
| 状态 | 图标 | 说明 |
|------|------|------|
| 主修 (Primary) | 🏆 | 您的主要专业，可教授该乐器所有课程 |
| 辅修 (Secondary) | ✓ | 具备教学能力，可作为辅助教学 |
| 助教 (Assistant) | ⏱ | 协助主修教师进行教学 |

### 申请新资质

1. 进入「教学资质」页面
2. 在「可申请资质」区域找到目标乐器
3. 点击「申请资质」按钮
4. 选择熟练程度：
   - **主修**: 您的专业乐器
   - **辅修**: 您具备教学能力的乐器
   - **助教**: 协助教学的乐器
5. 点击「确认申请」

> **注意**: 资质申请后需要教研室管理员审核。

### 撤销资质

1. 在「我的教学资质」区域找到要撤销的资质
2. 点击资质卡片上的「撤销」按钮
3. 确认操作

> **警告**: 撤销资质将影响相关课程的排课安排。

## 排课管理

### 基本排课规则

系统会自动验证以下规则：

1. **教研室匹配**: 教师只能教授其教研室对应的乐器课程
2. **资质验证**: 教师必须有相关乐器的教学资质
3. **时间冲突**: 同一教师同一时间只能有一节课
4. **班级规模**: 不能超过该乐器的最大班额

### 排课步骤

1. 进入「排课管理」页面
2. 选择学生（系统会自动关联其教研室）
3. 选择日期和节次
4. 系统会自动筛选符合条件的教师
5. 选择教师
6. 点击「确认排课」

### 跨教研室排课

默认情况下，系统不允许跨教研室排课。如果需要特殊情况处理：
1. 联系管理员开通权限
2. 或调整学生/教师的教研室归属

### 班级规模限制

| 乐器类型 | 最大班额 |
|----------|----------|
| 钢琴、聲乐、小提琴、萨克斯等 | 5人 |
| 古筝、笛子、竹笛、葫芦丝 | 8人 |

## 教研室视图

### 访问视图

左侧导航点击「教研室排课视图」

### 功能说明

#### 视图切换
- **周视图**: 显示一周所有课表
- **日视图**: 显示单日详细课表

#### 筛选功能
1. 按教研室筛选：点击左上角筛选器，选择教研室
2. 按乐器筛选：选择教研室后，可进一步按乐器筛选
3. 重置筛选：点击「重置」按钮

#### 课程卡片颜色
| 教研室 | 颜色 |
|--------|------|
| 钢琴专业 | 蓝色 |
| 声乐专业 | 绿色 |
| 器乐专业 | 橙色 |

### 操作说明
- 点击课程卡片可查看详情
- 右键菜单可进行调课、停课等操作

## 统计仪表板

### 访问仪表板

左侧导航点击「教研室统计」

### 统计内容

#### 总体统计
- **总课程数**: 本周期所有已排课时
- **教师总数**: 所有教研室的教师数量
- **学生总数**: 参与课程的唯一学生数量
- **课程总数**: 不同课程类型的数量
- **平均利用率**: 教室使用率

#### 各教研室详情
点击教研室卡片可展开查看：
- 课程数、教师数、学生数
- 教室利用率进度条
- 周环比变化趋势

#### 热门课程
显示各教研室最受欢迎的课程排名及趋势。

#### 教师工作量TOP5
显示工作量最高的前5名教师。

### 时间范围筛选
右上角可切换筛选周期：
- **本周**: 显示本周数据
- **本月**: 显示本月数据
- **本学期**: 显示学期累计数据

## 常见问题

### Q1: 为什么我不能给某学生排课？

可能原因：
1. 该学生已被其他教师预约该时段
2. 您不属于该学生对应的教研室
3. 您没有该乐器的教学资质
4. 班级已达到最大人数

### Q2: 如何添加新的乐器类型？

请联系系统管理员，管理员可以在「系统管理」中添加新的乐器配置。

### Q3: 误撤销了资质怎么办？

请联系教研室管理员，管理员可以在「教师管理」中恢复您的资质。

### Q4: 排课被拒绝，提示"教研室不匹配"

这是因为您不属于教授该乐器对应的教研室。
解决方案：
- 请管理员调整您的教研室归属
- 找该教研室的其他教师代课

### Q5: 提示"工作量过高"是什么意思？

当您在某一天排课超过8节时，系统会显示警告。建议：
- 适当减少该日排课
- 将部分课程调整到其他日期

### Q6: 如何查看我的工作量？

进入「个人中心」-「工作量统计」，可查看：
- 今日课时
- 本周课时
- 各教研室分布

### Q7: 可以跨教研室排课吗？

默认不允许。如有特殊情况：
1. 联系管理员开通跨教研室排课权限
2. 或请其他教研室的教师代课

### Q8: 班级规模如何计算？

班级规模按课程类型计算：
- **个别课**: 1对1教学
- **小组课**: 2-5人（钢琴等）或2-8人（古筝等）
- **合奏课**: 根据实际需要配置
