## 4. 使用说明

### 文件格式要求
- 支持格式：`.xlsx`, `.xls`, `.csv`
- 第一行必须是列标题（字段名）
- 编码格式：UTF-8（CSV文件特别注意）

### 导入步骤
1. 在对应管理页面点击「导入 Excel」按钮
2. 选择符合模板格式的 Excel/CSV 文件
3. 系统会自动解析并导入数据
4. 导入完成后会显示导入结果

### 注意事项
- 如果学生学号重复，系统会自动处理
- 课程类型和教室类型必须使用指定的关键词
- 数值字段会自动转换为数字格式
- 导入失败时会有错误提示，请检查文件格式
